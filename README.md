# Exchange Terminal Client

```bash
./bitwyre_client
```

Once connected, the user could then
```bash
1. Order Management System
  1. Make order
  2. Modify order
  3. Cancel order
  4. List orders
2. Account holdings
   1. Amount holdings (with wallet address)
   2. USD/CAD holdings
   3. Banking details (edited)
3. Account Settings
  1. Change Keys
  2. Change username
4. Exit
``` 

# Open Source Libraries
Pistache (REST API), Boost
